Button Module Cheat Sheet

Posted on Tuesday April 10, 2018 by Eric Potvin

Key Switch Module

  • Rating: 50mA 12VDC
  • Contact Resistance : 50mΩ max (initial)
  • Insulation Resistance: 100MΩ(minDC 250V)
  • Dielectric Strength : AC250V(50/60Hz for 1minute)
  • Electrically Life: 100,000cycles
  • Operating Force: 180 / 230(±20gf)

Keypad 4 Button Key Module Switch

  • Working voltage: DC 0 - 48V

Module Mcu 16 Keypad New Keyboard 4x4 Matrix

  • Size: Approx.42mm X 42mm / 1.65" X 1.65"

Push Button Matrix 16 Key Switch Keyboard

  • Size: 50mm x 60mm

Lockless ON/OFF Push Button Switch

  • Maximum voltage: 12V
  • Rated thermal current: 3(A)
  • Temperature range: -25° to 85°

Round Rocker Toggle Switches

  • Voltage: 250 v / 125 v
  • Current: 6a

Big Button Module For High Level Output

  • Voltage: 250 v / 125 v
  • Current: 6a
