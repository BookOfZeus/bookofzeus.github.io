Led Sensor Cheat Sheet

Posted on Thursday April 19, 2018 by Eric Potvin

DC 3V-5V 12 LED Super Bright White LED Piranha Board

  • Input voltage: DC 3.5-5.5V (Recommended 5V DC)
  • Operating current: about 200MA
  • Power: about 1W

Cathode Digital LED Kinescope Display Nixie Tube

  • Type : Common Anode
  • 1 digit 7 segment Display

Stick 8 x WS2812 5050 RGB LED

  • Each pixel of the three primary color can achieve 256 brightness display
  • Scan frequency not less than 400Hz/s
  • Send data at speeds of 800Kbps

7-Bit WS2812 5050 RGB LED Ring

  • Each pixel of the three primary color can achieve 256 brightness display
  • Scan frequency not less than 400Hz/s
  • Send data at speeds of 800Kbps

KY-016 RGB 3 Color Full Color LED

  • Use plug-in full-color LED
  • RGB tricolor limiting resistor to prevent burn
  • Operating voltage: 5V
  • LED drive mode: common cathode drive

KY-009 3 Colour RGB SMD LED Board Module

  • Using the 5050 full-color LED
  • RGB tricolor limiting resistor to prevent burn
  • Operating voltage: 5V
  • LED drive mode: common cathode drive

Multicolor 6 Color LED Matrix

  • 6 bit Multicolor Common Cathode LEDs
  • 1K ohm Limiting resistor
  • Power Supply Voltage: 3.3-12V

MAX7219 EWG 8-Digit Digital Tube Display Control

  • Integrated serial input/output common cathode
  • 64 independent LED

Dot-Matrix 8*8 LED Light Display

  • Type: common cathode Led
  • Forward Voltage (V) : 2.1 ~ 2.5
  • Reverse Current (uA) : <= 20
  • Max Continuous Forward Current(IFM): 20mA
  • Max Power Dissipation(PM): 40mW
  • Max Peak Forward Current(IFP): 100mA
  • Lead Soldering Temperature : 260 Degree (<5Sec)
  • Operating Temperature Range : -30 ~ +70 Degree
  • Preservative Temperature Range : -40 ~ +85 Degree

Building Blocks Full Color LED

  • Voltage: 5V
  • Input: Digital level

4P LED Module PWM dimming

  • Operating voltage: 3V to 5V
