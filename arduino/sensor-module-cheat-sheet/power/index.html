Power Sensor Cheat Sheet

Posted on Sunday April 08, 2018 by Eric Potvin

5V One Channel Relay Module

  • Maximum load: AC 250V/10A, DC 30V/10A
  • Trigger current: 5mA
  • Working voltage: 5V

5V One Channel Relay Module AC 120V DC 24V 2A

  • Maximum load: AC 120V/2A, DC 24V/2A
  • Trigger current: 5mA
  • Working voltage: 5V

MOS FET Driver Module

  • Voltage: 3.3V, 5V
  • Ports: Digital Level
  • Output load voltage :0-24V
  • Output load current: <5A (1A above need to add heat sink)

Level Conversion Module

  • AVCC connect to 5V system power
  • ASCL connect to 5V system SCL
  • ASDA connect to 5V system SDA
  • AGND connect to 5V system GND
  • BVCC connect to 3V system power
  • BSCL connect to 3V system SCL
  • BSDA connect to 3V systems SDA
  • BGND connect to 3V system GND.

DC 0~25V Voltage Sensor

  • Detect voltage (VDC) from 0 to 25V
  • VCC: 5V/3.3V

TTL Turn To RS485 Module Hardware Automatic Flow Control

  • 3.3V and 5.0V power perfect compatibility
  • 3.3V and 5.0V signal perfect compatibility
